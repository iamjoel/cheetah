# 工具
> 工欲擅其事，比先利其器

选择合适的工具。然后是好用工具。

![](./mind/tool.png)

## 合适的工具
最基本的：能完成任务。

进阶：高效的完成任务。体现：
1. 易用。用起来很直觉，没有心智负担。
2. 自动化。
   1. 模板：信息模板，工作流模板。
   2. 工作流。
      1. 事件触发。
      2. 时间触发。
3. 可拓展性。
4. 运行速度。
